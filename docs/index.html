Bin the sites you're trying to quit.

A browser extension that tosses the sites you keep slipping back to into bins — then warns you, redirects you, or replaces them with a stop page. Install

  1. Download the ZIP above and unzip it somewhere permanent — not your Downloads folder.
  2. Open chrome://extensions (or brave://, edge://, vivaldi://extensions).
  3. Turn on Developer mode (top-right toggle).
  4. Click Load unpacked and pick the unzipped folder — the one with manifest.json.
  5. Pin the icon, open it, make a bin, and paste in a list. Grab a preset ↓

It loads as a developer/unpacked extension and stays installed across restarts. One-click store versions are TBA — see the chips above.

Presets

Curated lists, ready to paste into a bin. Copy one, make a new bin in the extension, hit Paste domains, and you're done.

Got your own pile to take out? Contributions are very welcome. Drop a .txt file (one domain per line) into the presets/ folder and open a pull request — we'll add it here.
